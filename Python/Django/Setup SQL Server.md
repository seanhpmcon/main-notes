### Pre-requisite for Linux
For sql server to work, the sql server odbc driver and unixodbe needs to be installed.
See Microsoft site to install -> https://learn.microsoft.com/en-us/sql/connect/odbc/linux-mac/installing-the-microsoft-odbc-driver-for-sql-server?view=sql-server-ver16&tabs=alpine18-install%2Calpine17-install%2Cdebian8-install%2Credhat7-13-install%2Crhel7-offline
