# Functional
## Desktop Version
### Product Page
1. Must display an image of the Item to the left that can be scrolled through
2. Must display a description to the right.
3. Must have button to add Item to cart
4. Must have a navigation outline at the top to show where the product is located

### List of Items Page
1. Must List items in Product card
2. Must display description of item when hover
3. must be able to filter, sort
4. Must have a navigation filter to the left
5. Must show filters to the top of page
6. Navigation filter and products page must have navigation in its own divs

### Deliveries
1. Must have a calendar to select dates for deliveries
2. Must have a Map to select location for deliveries
