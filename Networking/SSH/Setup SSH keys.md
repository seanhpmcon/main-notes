## This is beneficial for password-less login

### SSH key location
SSH keys are located in the .ssh/aithorized_keys repo where all the public keys of remote users are stored so they can access you git repo.

### Server IP address
This is the general structure of the address `user@ipaddress` with example `git@192.168.1.111`
