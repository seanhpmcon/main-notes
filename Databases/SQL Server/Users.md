When creating a new user, ensure that the 'deny' permissions of the user are un checked to prevent the following error:
```
# [The SELECT permission was denied on the object 'sysobjects', database 'mssqlsystemresource', schema 'sys'](https://stackoverflow.com/questions/1473315/the-select-permission-was-denied-on-the-object-sysobjects-database-mssqlsyst)
```
